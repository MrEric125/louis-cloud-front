项目目录分为这么几个模块(管理员)
1. 首页
2. 权限设置
    - 用户管理 包括（CRUD,以及批量操作功能）
    - 菜单管理 也就是某个角色能够看到的菜单
    - 权限管理 基于RBAC 模型的操作
    - 角色 基于RBAC 模型
3. 商品
    - 商品列表
    - 添加商品
    - 商品分类
    - 品牌管理
4. 订单
    - 订单列表
    - 订单设置
    - 退货申请处理
    - 退货原因设置
5. 营销
    - 秒杀活动列表
6. 运营监控
    - 数据库监控
    - 接口调用监控
    - 登录信息信息监控
    - 日志监控
